Aplicativo de estudo **single-file** (um único `index.html`, sem instalação e sem servidor) para aprender a linguagem **C** de forma estruturada, com base na disciplina **Programação Estruturada** (IFRS Campus Canoas).

> Funciona online no link acima, ou baixe e abra o `index.html` no navegador — tudo funciona offline e seus dados ficam salvos no próprio navegador (`localStorage`).

---

## ✨ Funcionalidades

### 🧭 Trilha do Conhecimento
Fluxograma animado que percorre os 23 módulos, do primeiro `printf` ao projeto final, com tour guiado.

### 🗺️ Roadmap completo
23 módulos com status (a estudar → estudando → dominado), avaliação de dificuldade e anotações por tópico.

### 🎓 Apostila teórica
Aula explicativa para cada módulo, com seções, caixas de atenção/dica, código e resumo. Leitor com navegação e marcação de "lida".

### 💻 Exercícios reais
~80 exercícios das listas da disciplina, com editor de código embutido, botão para rodar no compilador online, autoavaliação e registro de dificuldade.

### 💡 Exemplos por nível
Exemplos de código em 3 níveis (Básico / Intermediário / Avançado), cada um com o **raciocínio passo a passo** de como foi desenvolvido.

### 🧪 Revisar (ciência do aprendizado)
- **Flashcards** (~165) com **repetição espaçada** (algoritmo SM-2)
- **Quizzes** (~70 questões) de múltipla escolha
- **Simulado cronometrado**
- Painel "Revisão de hoje"

### 🏆 Gamificação e controle
XP e níveis · sequência de dias (streak) 🔥 · conquistas/badges · mapa de atividade (heatmap) · metas semanais · timer **Pomodoro** · estatísticas.

### 🎯 Mapa de Dificuldades
Identifica automaticamente onde você mais trava, cruzando roadmap, exercícios e autoavaliação.

## 🛠️ Como usar

1. Baixe ou clone o repositório.
2. Abra `index.html` no navegador (duplo clique).
3. Estude! Marque progresso, resolva exercícios e revise com flashcards.
4. Use o botão **⬇ Backup** de tempos em tempos para exportar seus dados.

## 💾 Backup dos dados

Os dados ficam no `localStorage` do navegador. Para não perder, exporte um `.json` pelo botão de backup na barra lateral e guarde-o. O botão de restaurar recarrega tudo.
